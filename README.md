# Jelly Voices
---
This is a mod that adds various voices to SPT. Over time as i find good clips I'll flush out the voices more.

## Voice List
- Jerma985
- Hunk (RE9 & RE:Operation Raccoon City)
- Leon (RE4 remake)
- More to come!

## Uninstalling
Before uninstalling make sure you change your voice back to a default. Then you can safely delete the "**JellyVoices**" folder in your "**SPT/user/mods**".
